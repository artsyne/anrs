# Backlog

Future tasks and technical debt waiting to be prioritized.

## Usage

1. Add new task ideas here
2. Prioritize and refine before promoting to `active/`
3. Use `task-template.md` as starting point

## Promoting to Active

```bash
cp .anrs/plans/templates/task-template.md .anrs/plans/active/task-{id}.md
# Edit task details
# Update .anrs/state.json with current_task
```
