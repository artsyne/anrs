# Failure Cases

Archived failure cases from harness evaluations for analysis and improvement.

## Purpose

- Learn from past failures
- Identify recurring patterns
- Improve harness detection rules
- Train AI on edge cases

## File Format

```markdown
# Failure Case: FC-{id}

## Context
- Task: {task_id}
- Skill: {skill_name}
- Harness Level: L1/L2/L3

## Error
{error_message}

## Root Cause
{analysis}

## Fix Applied
{solution}

## Prevention
{how_to_prevent}
```

## Categories

| Category | Description |
|----------|-------------|
| `syntax/` | Syntax and lint errors |
| `test/` | Test failures |
| `security/` | Security violations |
| `stability/` | Stability issues |
