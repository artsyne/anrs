# ANRS VS Code Extension

VS Code extension for ANRS-compliant repositories.

## Status

**Planned** - Not yet implemented.

## Planned Features

- Detect ANRS repositories (`.anrs/` directory)
- Show current state in status bar
- Tree view for plans (active/backlog/completed)
- Quick commands:
  - `ANRS: Show State`
  - `ANRS: Run Harness`
  - `ANRS: View Plan`
- Syntax highlighting for ANRS files

## Development

```bash
cd ext/vscode
npm install
npm run compile
```
