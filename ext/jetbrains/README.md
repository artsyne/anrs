# ANRS JetBrains Plugin

JetBrains IDE plugin for ANRS-compliant repositories.

## Status

**Planned** - Not yet implemented.

## Planned Features

- Detect ANRS repositories (`.anrs/` directory)
- Tool window for state and plans
- Run configurations for harness
- Inspections for ANRS compliance

## Development

Uses Gradle-based IntelliJ Platform Plugin development.
